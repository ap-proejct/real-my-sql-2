## 📋 스터디 규칙

### ✅ 기본 규칙

1. **매주 1챕터** 분량을 읽고 각자 README.md에 정리합니다.
2. 정리는 **스터디 전날 자정(24:00)** 까지 PR을 올립니다.
3. 스터디 당일에 서로의 정리를 **리뷰**하고 질문/토론합니다.
4. 모르는 내용은 솔직하게 질문하고, 아는 내용은 적극적으로 공유합니다.

### 📝 정리 방법

- 각자 본인 이름 폴더(`정주연/`, `곽유섭/`) 안의 `README.md`에 정리합니다.
- 이미지(캡처, 다이어그램 등)는 `images/` 폴더에 넣고 README에 삽입합니다.
- 정리 형식은 자유이나, 핵심 개념과 본인 생각은 반드시 포함합니다.

### 🔀 PR & 머지 규칙

1. 브랜치 네이밍: `chapter01/정주연`
2. PR 제목: `[Chapter 01] 정주연 - 정리 완료`
3. 상대방의 PR을 확인 후 **Approve** 해야 머지 가능합니다.
4. 스터디 종료 후 `main` 브랜치에 머지합니다.

### ⏰ 스터디 일정

- **요일**: 매주 협의 (Discord/카카오톡 공지)
- **시간**: 약 1~2시간
- **방식**: 온/오프라인 병행
